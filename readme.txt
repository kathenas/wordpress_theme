== Description ==

Kathenas is a minimalistic and elegant WordPress theme featuring an ultra clean magazine layout. With a beautiful typography, various post layouts and a gorgeous featured posts slideshow Kathenas truly helps you to stand out.

For more information about Kathenas please go to https://development.kathenas.org/internal/website/themes/kathenas/
